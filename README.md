# OffWorkYet?

> A tiny website for two tired people who just want to go home.

---

## What is this?

**OffWorkYet** is a minimal web page for couples (or coworkers) who want to sync their off-work status  
**without sending messages, calling, or interrupting meetings**.

Because sometimes:
- you're busy,
- they're busy,
- and everyone is pretending to work.

## Tech Stack

- Pure Frontend (HTML + Tailwind CSS)
- Supabase (Free tier, no backend server)
- GitHub Pages deployment
- No build step
- No login
- No tracking
- No ambition

## Why?

Because love should be:
- asynchronous
- low-interruption
- and slightly sarcastic

This project:
